Sarcasm AI

Ever miss a sarcasm memo? Not anymore! SarcasmSense™, our AI wingman, is here to decode the witty banter for you. No more eyebrow raises or confused nods – just pure, unfiltered humor translation. Keep it snarky, stay sharp, and let SarcasmSense™ be your guide to navigating the sarcasm maze. Because life's too short to miss the joke!

Prerequisites:

Dependencies:

This application requires Docker to be installed on your machine. If you don't have Docker installed, you can download and install it from the official Docker website:

Download Docker

Installation:

The application is meant to be self-hosted, hence to run it, you can download the Sarcasm AI archive from the following link:

SarcasmAi Download

How to run:

Once the application archive is downloaded, it should be extracted. Now, in the console of your choice, go to the extracted directory.

The folder has two bootstrapping script files:

  • bootstrap (UNIX)
  • bootstrap.bat (Windows)
To run the application, one has to execute a bootstrapping script and wait for the application to deploy. Once you see an IP-address with a port 5000 (e.g. 172.17.0.2:5000) it means the app is successfully deployed and accesisble via that link.
